# CLAUDE.md

This file provides guidance to Claude Code when working with code in this repository.

## What This Is

Scalar Sandbox — a full-stack app for sharing and previewing OpenAPI documents. Vue 3 frontend with Monaco editor + Scalar API Reference viewer, backed by Cloudflare Workers (Pages Functions) and D1 (SQLite).

## Commands

```bash
pnpm install              # Install dependencies
pnpm dev                  # Run frontend (Vite) + backend (Wrangler) concurrently
pnpm frontend             # Vite dev server only
pnpm backend              # Wrangler Pages dev server on port 8788
pnpm build                # Type-check (vue-tsc) then Vite build
pnpm format               # Format with Prettier
pnpm format:check         # Check formatting
pnpm db:generate          # Generate Drizzle migrations from schema
pnpm db:migrate           # Apply migrations locally
pnpm db:migrate:production # Apply migrations to production D1
```

No test framework is configured.

## Architecture

**Not a monorepo.** Single package, full-stack:

- `src/` — Vue 3 SPA (Vue Router, Monaco Editor, Scalar themes/parser)
- `functions/` — Cloudflare Pages Functions (serverless API)
  - `api/share.ts` — POST to create/update specs
  - `api/share/[id].ts` — GET spec by ID
  - `files/[[id]].ts` — GET spec as openapi.json or openapi.yaml
- `db/schema.ts` — Drizzle ORM schema (single `Specs` table on D1/SQLite)
- `migrations/` — SQL migrations generated by Drizzle Kit

**Dev proxy:** Vite proxies `/api/*` and `/files/*` to Wrangler on port 8788 during development.

## Key Conventions

- **Formatting:** Prettier with trailing commas, single quotes, 2-space indent, no semicolons
- **Linting:** ESLint 9 flat config with Vue and Prettier plugins (`src/.eslintrc.js`)
- **TypeScript:** Strict mode, ES2020 target, bundler module resolution
